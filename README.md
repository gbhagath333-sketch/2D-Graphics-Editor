# 2D Graphics Editor

A menu-driven 2D Graphics Editor developed in C using the ncurses library.

## Features
- Draw Point
- Draw Line (Bresenham Algorithm)
- Draw Circle (Midpoint Circle Algorithm)
- Draw Rectangle
- Display Canvas
- Clear Canvas
- Save and Load Drawings
- Undo and Redo Operations

## Technologies Used
- C Programming
- ncurses Library
- Git & GitHub

## Compilation

gcc graphics_editor.c -o graphics_editor -lncurses -lm

## Execution

./graphics_editor
